ex009

Listas

Para usar lista é usado <ol> (para lista ordenadas) e <ul> (para listas não ordernadas) sendo usado os parâmetros:

type="" para escolher o tipo de representação da lista. ex: 1, A, a, I, i. (somente listas ordenadas) ou disc, square ou circle. (somente listas não ordenadas)

start="" para escolher em qual posição a lista vai começar (somente ol)

ex017

Fontes

Medidas

Medidas absolutas:
cm, mm, in, px, pt, pc

Medidas relativas:
em, ex, rem, vm, vh, %

Recomendação da W3C:
px, em

16px - tamanho padrão da fonte
1em - tamanho padrão da fonte
2em - dobro do tamanho padrão (e assim por diante)

Shorthand

É possível usar o atributo 'font' como shorthand.
sintáxe: font: {font-style} {font-weight} {font-size} {font-family}

Fontes externas


    @font-face {
        font-family: 'London'; /*Nome da fonte importada*/
        src: url(Olondon_.otf), url(OldLondon.ttf); /*Localização*/
        font-weight: normal; /*Tamanho*/
        font-style: normal; /*Estilo*/
        /*
            Tipos de format()
            - opentype (quando é otf);
            - truetype (quando é ttf);
            - embedded-opentype;
            - truetype-aat (Apple Advanced Typography)
            - svg
        */
    }

ex020

decoração de links

usa-se a pseudoclasse "::before" e "::after" para decorar links antes e depois. ex: 


    a::before {
        content: '~';
        text-decoration: none;
        font-weight: normal;
        color: darkgrey;
    }
    a::after {
        content: '[link] ';
        text-decoration: none;
        font-weight: normal;
        color: darkgrey;
    }

ex022

background em css

o elemento background tem os seguintes atributos:
background-color: black;
background-image: url('imagens/wallpaper002.jpg');
background-position: center center;
background-repeat: no-repeat;
background-size: cover;
background-attachment: fixed;

Que podem ser usados em maneira menor com uma shorthand:
background: {color} {image} {position} {repeat} {size} {attachment};
